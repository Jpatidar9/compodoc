OutputGroupType

Represents the type of output group for media conversion. It can be one of the following:

  • "HLS_GROUP": HLS output group.
  • "DASH_ISO_GROUP": DASH ISO output group.
  • "FILE_GROUP": File output group.
  • "MS_SMOOTH_GROUP": Microsoft Smooth Streaming output group.
  • "CMAF_GROUP": CMAF output group.
  • string: Any other custom output group type.
"HLS_GROUP" | "DASH_ISO_GROUP" | "FILE_GROUP" | "MS_SMOOTH_GROUP" | "CMAF_GROUP" | string
